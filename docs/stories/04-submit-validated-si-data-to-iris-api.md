# User Story: 4 - Submit Validated SI Data to IRIS API

**As an** external customer,
**I want** to submit my validated SI data to the IRIS API after confirming it's correct,
**so that** my shipping instructions are processed and integrated into the system.

## Acceptance Criteria

- User can review all SI data before final submission
- Clear confirmation action is required before submission
- System submits validated SI data to the IRIS API
- User receives confirmation when submission is successful
- Submission status is clearly displayed during the process
- User can see a summary of what was submitted

## Notes

- Confirmation step should provide a final review opportunity
- Consider showing a summary or preview before final submission
- Successful submission should provide a reference number or confirmation ID
